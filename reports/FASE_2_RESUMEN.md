# Distrito Goo · Fase 2

## Cambios aplicados

- Experiencia responsive reforzada para escritorio, tablet y móvil.
- Tarjeta del District Coach reducida a la fotografía como acceso, sin textos redundantes.
- Herramientas con vistas Todas, Favoritos y Recientes, además de orden por CMS, nombre o uso.
- Preferencias, favoritos, recientes y uso almacenados localmente.
- Persistencia de la última sección visitada durante la sesión.
- Estado offline visible y actualización de PWA voluntaria mediante el botón **Actualizar**.
- Foco visible, reducción de movimiento y controles táctiles de al menos 40–44 px.
- Pipeline Python separado en validación, compilación y auditoría de enlaces.

## Compatibilidad

La aplicación continúa siendo estática. Python se utiliza sólo antes de publicar. No se agregó backend, no se alteraron enlaces del CMS y GitHub Pages sigue usando rutas relativas.
